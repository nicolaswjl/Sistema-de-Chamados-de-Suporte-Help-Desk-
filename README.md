#  Sistema de Chamados de Suporte

Este projeto consiste em um sistema simples de gerenciamento de chamados de suporte técnico, desenvolvido em Python, com foco em lógica de programação e resolução de problemas.

---

##  Funcionalidades

-  Abrir chamado
-  Listar chamados
-  Marcar chamado como resolvido
-  Excluir chamado

---

##  Tecnologias utilizadas

- Python

---

##  Objetivo

O objetivo deste projeto é praticar conceitos fundamentais de programação, como:

- Estruturas condicionais
- Laços de repetição
- Listas e dicionários
- Organização de dados

---

##  Como executar

1. Certifique-se de ter o Python instalado
2. Baixe ou clone este repositório
3. Execute o arquivo:

```bash

=== SISTEMA DE CHAMADOS ===
1 - Abrir chamado
2 - Listar chamados
3 - Resolver chamado
4 - Excluir chamado
5 - Sair

python nome_do_arquivo.py
